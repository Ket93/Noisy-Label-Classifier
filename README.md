# Noisy-Label-Classifier

## Setup

1. Download the dataset from Kaggle:
   `pip install kaggle`
   `kaggle datasets download -d alxmamaev/flowers-recognition`

2. Unzip and place it at:
   mylibs/data/flowers/
   
3. Structure should look like:
   mylibs/data/flowers/
   ├── daisy/
   ├── dandelion/
   ├── rose/
   ├── sunflower/
   └── tulip/
